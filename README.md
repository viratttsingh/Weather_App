## Overview

The **Weather Forecast Application** is a simple, user-friendly web application that allows users to view real-time weather and geographical information for any city worldwide. It leverages the **OpenWeather API** to fetch accurate and up-to-date weather data, presenting it in a clean, structured, and easy-to-read format.

* **Real-Time Weather Data**: Fetches current weather information for any city using the OpenWeather API.

  * UV index
  * Day/Night status
* **Location Information**: Shows geographical and time-related details, including:

  * City and country
  * Current time and date
  * Latitude and longitude
  * Time zone
* **Responsive Design**: Ensures a smooth and consistent user experience across desktops, tablets, and mobile devices.
